Conectar FinanzasBach con tu página Wix

Guía paso a paso para incrustar esta app en tu sitio Wix. Tienes dos métodos: el más rápido usa GitHub Pages gratis para alojar el archivo, y el más avanzado usa Wix Velo para integración nativa.

1
Descarga el archivo de la app

Descarga finanzas-bachillerato.html (el archivo que estás usando ahora mismo) y guárdalo en tu computadora.

ARCHIVO finanzas-bachillerato.html
Tamaño aprox. 40 KB · Solo HTML, sin dependencias externas críticas
2
Crea un repositorio en GitHub y activa GitHub Pages
2a · Crear cuenta y repositorio
  1. Entra a github.com y crea una cuenta gratuita
  2. Haz clic en el botón verde New (arriba a la izquierda)
  3. En Repository name escribe: finanzasbach
  4. Selecciona Public (es obligatorio para Pages gratis)
  5. Marca Add a README file
  6. Clic en Create repository
2b · Subir el archivo HTML
  1. Dentro del repositorio haz clic en Add file → Upload files
  2. Arrastra el archivo finanzas-bachillerato.html
  3. En la parte inferior escribe en el mensaje: "Subir app FinanzasBach"
  4. Clic en Commit changes
2c · Activar GitHub Pages
  1. Ve a la pestaña Settings del repositorio
  2. En el menú lateral busca Pages
  3. En Source selecciona Deploy from a branch
  4. En Branch elige main → carpeta / (root)
  5. Clic en Save
  6. Espera 2–3 minutos y recarga la página
✅ Tu URL pública quedará así:
https://TU_USUARIO.github.io/finanzasbach/finanzas-bachillerato.html
Reemplaza TU_USUARIO por tu nombre de usuario de GitHub.
3
Insertar la app en el editor Wix
3a · Abrir el editor Wix
  1. Entra a manage.wix.com
  2. Selecciona tu sitio y haz clic en Editar sitio
  3. En la página donde quieras mostrar la app, haz clic en ＋ Añadir elementos (barra lateral izquierda)
  4. Ve a Incrustado → HTML personalizado (también aparece como Embed)
  5. Arrastra el bloque a la página
3b · Configurar el bloque HTML
  1. Haz doble clic sobre el bloque HTML recién añadido
  2. Selecciona Introduce tu código
  3. Borra todo el contenido que aparezca
  4. Pega el código iframe de abajo
  5. Haz clic en Actualizar
  6. Ajusta el tamaño del bloque arrastrando sus bordes (recomendado: 100% ancho, 800px alto)
📋 Código a pegar en Wix — reemplaza TU_USUARIO:
<iframe src="https://TU_USUARIO.github.io/finanzasbach/finanzas-bachillerato.html" width="100%" height="800" frameborder="0" allow="clipboard-write" style="border:none; border-radius:16px; display:block;" title="FinanzasBach — App de finanzas personales para bachillerato" ></iframe>
⚠️ Nota importante sobre Wix
Wix no permite iframes de todos los dominios en su editor. Si el iframe no carga, ve a Configuración del sitio → Permisos de contenido y agrega github.io como dominio de confianza. Alternativamente, usa el Método B (Wix Velo).
4
Añadir botón de WhatsApp flotante en Wix
Opción A · Botón nativo de Wix
  1. En el editor, ve a ＋ Añadir → Botón
  2. Elige el estilo que prefieras
  3. Escribe el texto: "Soporte WhatsApp"
  4. En Enlace selecciona URL externa
  5. Pega la URL de abajo y marca Abrir en nueva pestaña
  6. Fija el botón a la página con Fijar al scroll (esquina inferior)
https://wa.me/525523239785?text=Hola,%20necesito%20soporte%20con%20FinanzasBach%20%F0%9F%92%B0
Opción B · Widget oficial de WhatsApp para Wix
  1. Ve al Wix App Market
  2. Busca "WhatsApp Chat"
  3. Instala la app gratuita WhatsApp Chat by POWR o Elfsight WhatsApp Chat
  4. En la configuración ingresa el número: 525523239785
  5. Activa la burbuja flotante y personaliza el mensaje de bienvenida
  6. Publica los cambios
5
Crear una sección explicativa en Wix sobre tu proyecto

Tu página Wix puede tener una sección de presentación antes del iframe. Aquí un ejemplo de estructura recomendada:

📌 Sección Hero
Título del proyecto, logo FinanzasBach, y una breve descripción de 2–3 líneas. Añade un botón "Usar la app" que lleve al anchor del iframe.
📌 Sección "¿Qué hace?"
3 tarjetas con íconos: "Registra ingresos y gastos", "Analiza con gráficas", "Aprende a invertir". Wix tiene el componente Grid de íconos listo para usar.
📌 Sección App (iframe)
El bloque HTML de Wix con el iframe de la app. Ponlo en sección de fondo oscuro para que resalte bien visualmente.

¿Qué es Wix Velo?

Wix Velo es el entorno de desarrollo integrado de Wix. Permite escribir JavaScript para crear páginas dinámicas, conectar bases de datos y controlar elementos de la página directamente desde el editor. Es el método más potente para una integración real.

A
Activar Wix Velo en el editor
  1. Abre el editor Wix de tu sitio
  2. En el menú superior haz clic en Dev Tools (o Herramientas de desarrollo)
  3. Selecciona Activar Velo y confirma
  4. Aparecerá un panel de código en la parte inferior del editor
B
Crear la base de datos en Wix CMS

Con Wix CMS puedes guardar los datos financieros en la nube, accesibles desde cualquier dispositivo.

  1. En el panel lateral izquierdo del editor, busca CMS → Bases de datos
  2. Clic en ＋ Nueva colección
  3. Nombre: Movimientos
  4. Agrega los campos:
Campo Tipo Descripción
tipo Texto "income" o "expense"
categoria Texto Categoría del movimiento
descripcion Texto Descripción del movimiento
monto Número Monto en MXN
fecha Fecha Fecha del movimiento
Establece los permisos de la colección en Solo el sitio puede escribir / Cualquiera puede leer (o requiere login si quieres usuarios privados).
C
Código Velo para guardar y leer movimientos

Copia este código en la pestaña Página del panel Velo de la página donde está la app:

// ─── FinanzasBach · Wix Velo Integration ─── import wixData from 'wix-data'; $w.onReady(function () { // Cargar movimientos al iniciar cargarMovimientos(); }); // Guardar un nuevo movimiento en el CMS export async function guardarMovimiento(tipo, categoria, descripcion, monto, fecha) { const toInsert = { tipo: tipo, // "income" o "expense" categoria: categoria, descripcion: descripcion, monto: parseFloat(monto), fecha: new Date(fecha) }; try { const resultado = await wixData.insert('Movimientos', toInsert); console.log('Movimiento guardado:', resultado._id); return resultado; } catch (err) { console.error('Error al guardar:', err); } } // Leer todos los movimientos ordenados por fecha export async function cargarMovimientos() { try { const resultados = await wixData.query('Movimientos') .descending('fecha') .limit(100) .find(); return resultados.items; } catch (err) { console.error('Error al cargar:', err); return []; } } // Eliminar un movimiento por su ID export async function eliminarMovimiento(id) { try { await wixData.remove('Movimientos', id); console.log('Movimiento eliminado:', id); } catch (err) { console.error('Error al eliminar:', err); } } // Obtener resumen del período (weekly/biweekly/monthly) export async function resumenPeriodo(periodo) { const ahora = new Date(); const dias = periodo === 'weekly' ? 7 : periodo === 'biweekly' ? 15 : 30; const inicio = new Date(ahora); inicio.setDate(ahora.getDate() - dias); const resultados = await wixData.query('Movimientos') .ge('fecha', inicio) .find(); let ingresos = 0, gastos = 0; resultados.items.forEach(item => { if (item.tipo === 'income') ingresos += item.monto; else gastos += item.monto; }); return { ingresos, gastos, balance: ingresos - gastos }; }
D
Publicar y probar el sitio
  1. En el editor, haz clic en Publicar (botón azul arriba a la derecha)
  2. Confirma la publicación y espera a que se complete
  3. Abre tu sitio en el navegador con la URL de Wix
  4. Prueba agregar un movimiento y verifica que aparezca en CMS → Bases de datos → Movimientos
  5. Revisa la consola del navegador (F12) si algo no funciona
💡 Consejo de depuración
Wix Velo tiene un Panel de registros en la parte inferior del editor. Ahí puedes ver los console.log en tiempo real mientras navegas el sitio en vista previa.
Comparativa de métodos
Método A — GitHub Pages + iframe
✅ Más fácil de configurar
✅ No requiere conocimientos de programación
✅ 100% gratuito
✅ Listo en menos de 30 minutos
⚠️ Los datos se guardan solo en el navegador del usuario
⚠️ Puede haber restricciones de iframe en Wix
Método B — Wix Velo + CMS
✅ Datos en la nube (acceso desde cualquier dispositivo)
✅ Integración nativa con Wix
✅ Posibilidad de múltiples usuarios
✅ Panel CMS para ver los datos
⚠️ Requiere conocimientos básicos de JavaScript
⚠️ Wix CMS tiene límites en plan gratuito
